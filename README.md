# Get Started
1. Clone:
   ```
   git clone https://github.com/plilab/notes.git
   ```
2. Install dependencies
   ```
   cd notes
   npm i
   ```
3. Show preview
   ```
   npm run build
   npm run preview
   ```
4. Look through the guides on writing. When you're about to start writing stuff, you can run the development server instead:
   ```
   npm run dev
   ```
